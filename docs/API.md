# API 文档

## 概览

四会项目后端API接口文档。

## 基础信息

- **基础URL**: `http://localhost:8080/api`
- **认证方式**: JWT Token
- **数据格式**: JSON

## 认证

所有需要认证的接口都需要在请求头中包含JWT token：

```
Authorization: Bearer <jwt_token>
```

## 接口列表

### 基础接口

#### 健康检查
```
GET /health
```

**响应示例:**
```json
{
  "status": "ok",
  "timestamp": "2024-01-01T00:00:00Z"
}
```

## 错误码

| 错误码 | 说明 |
|--------|------|
| 200    | 成功 |
| 400    | 请求参数错误 |
| 401    | 未授权 |
| 403    | 禁止访问 |
| 404    | 资源未找到 |
| 500    | 服务器内部错误 |
